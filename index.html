Aplicativo para generación de magnetita

Ana Lucía Campaña, Sergio Flórez, Juliana Noguera, Johann Osma

Universidad de los Andes. Grupo de investigación CMUA.

Este aplicativo tiene como fin estimar la cantidad de reactivos necesarios para la síntesis de una cantidad específica de magnetita.

Reactivos a usar son:

  • Cloruro Ferroso Tetrahidratado - \(FeCl_{2}\ \cdot 4H_{2}O \), peso molecular = 198.81 g/mol
  • Cloruro Ferrico Hexahidratado - \(FeCl_{3}\ \cdot 6H_{2}O \), peso molecular = 270.30 g/mol
  • Hidróxido de sodio - \(NaOH\), peso molecular = 39.997 g/mol

Reacción química de la síntesis de la magnetita:

\( Fe^{+2}Cl_{2}^{-1} + 2Fe^{+3}Cl_{3}^{-1} + 8Na^{+1}O^{-2}H^{+1} \rightarrow Fe^{+2}Fe_{2}^{+3}O_{4}^{-2} + 8Na^{+1}Cl^{-1} + 4H_{2}^{+1}O^{-2}\)

Peso molecular magnetita: 231.533g/mol

¿Qué cantidad de magnetita (\(Fe_{3}O_{4}\)) desea [g]?



Necesitará:


Gramos de \(FeCl_{2} \cdot 3H_{2}O\)
g
\(X_{g}(\frac{1 \ mol \ Fe_{3}O_{4}}{231.533 \ g}) (\frac{1 \ mol \ FeCl_{2}}{1 \ mol \ Fe_{3}O_{4}}) (\frac{1 \ mol \ FeCl_{2} \cdot 4H_{2}O}{1 \ mol \ FeCl_{2}}) (\frac{198.81 \ g \ FeCl_{2} \cdot 4H_{2}O}{1 \ mol \ FeCl_{2} \cdot 4H_{2}O})\)



Gramos de \(FeCl_{3} \cdot 6H_{2}O\)
g
\(X_{g}(\frac{1 \ mol \ Fe_{3}O_{4}}{231.533 \ g}) (\frac{2 \ mol \ FeCl_{3}}{1 \ mol \ Fe_{3}O_{4}}) (\frac{1 \ mol \ FeCl_{3} \cdot 6H_{2}O}{1 \ mol \ FeCl_{3}}) (\frac{270.30 \ g \ FeCl_{3} \cdot 6H_{2}O}{1 \ mol \ FeCl_{3} \cdot 6H_{2}O})\)



Gramos de \(NaOH\)
g
\(X_{g}(\frac{1 \ mol \ Fe_{3}O_{4}}{231.533 \ g}) (\frac{8 \ mol \ NaOH}{1 \ mol \ Fe_{3}O_{4}}) (\frac{39.997 \ g \ NaOH}{1 \ mol \ NaOH}) \)



Cada reactivo debe ser disuelto por aparte en \(H_{2}O \ mili \ Q\) con los siguientes volúmenes:



Mililitros de \(H_{2}O \ mili \ Q \ (Sol. 1M) \ \rightarrow \ FeCl_{2} \cdot 4H_{2}O\)
mL
\(X_{g}(\frac{1 \ mol \ Fe_{3}O_{4}}{231.533 \ g}) (\frac{1 \ mol \ FeCl_{2}}{1 \ mol \ Fe_{3}O_{4}}) (\frac{1 \ mol \ FeCl_{2} \cdot 4H_{2}O}{1 \ mol \ FeCl_{2}})\) \( (\frac{1000 \ mL \ H_{2}O }{1 \ mol \ FeCl_{2} \cdot 4H_{2}O}) \)
